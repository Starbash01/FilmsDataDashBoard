# FilmsDataDashBoard
Power BI Film Industry Dashboard built using SQL and Power BI to analyze movie revenue, release years, top-producing countries, language distribution, and film personnel. The dashboard delivers interactive insights into film production trends and industry performance through dynamic visualizations.

### Questions KPIs
1. What is the total gross revenue generated by the selected films?
2. Which film generated the highest gross revenue?
3. Which countries produce the most films?
4. What are the top five film-producing countries?
5. Which language is used most frequently in films?
6. What are the top three languages by number of films?
7. Which individuals appear most frequently in film roles?
8. What roles are most commonly represented in the dataset?
9. What trends can be observed in film production?
10. Which countries contribute most to the global film industry?
11. How do language preferences influence film production?

### Process Followed to Develop the Dashboard
* Defined the Project Objective
  * Established the goal of analyzing film industry data to identify trends in revenue, production, languages, and personnel.

* Extracted Data Using SQL

  * Queried the database to retrieve:
     * Film titles, gross revenue, and release years.
     * Top-producing countries by film count.
     * Most common film languages.
     * Film personnel and their associated roles using SQL joins.

* Imported Data into Power BI

     * Connected Power BI to the SQL database.
     * Imported the SQL query results into Power BI.
 
* Data Cleaning and Transformation
     * Used Power Query to remove inconsistencies, verify data types, and prepare the data for analysis.
 
* Developed DAX Measures

     * Created measures such as:
     * Total Gross Revenue
     * Total Films
     * Total Language
 
* Designed Interactive Visualizations

     * Built charts and KPI cards to display:
       * Gross revenue by film.
       * Total Revenue.
       * Top-producing countries.
       * Language distribution.
       * Film personnel and roles.
      
* Validated Dashboard Outputs

     * Compared dashboard results with SQL query outputs to ensure data accuracy.

### Outcome

The dashboard successfully integrates SQL data extraction with Power BI visualization to provide a clear overview of film industry performance. It enables users to explore revenue trends, production by country, language distribution, and personnel involvement, supporting data-driven analysis of the film industry.

### Key Insights

* A small number of countries account for the majority of film production, highlighting major contributors to the global film industry.
* English (or the leading language in the dataset) dominates film production, reflecting its widespread use in international cinema.
* Gross revenue varies significantly across films, with a few blockbuster titles contributing a large share of total earnings.
* Film production trends over the years reveal periods of increased and decreased industry activity.
* The personnel analysis identifies individuals and roles that frequently contribute to film productions, illustrating collaboration patterns within the industry.
* Integrating SQL with Power BI streamlined data extraction and enabled interactive exploration of key industry metrics.

### Project Conclusion
This project demonstrates the integration of SQL for data extraction and Power BI for interactive reporting and visualization. By analyzing film revenue, production trends, countries, languages, and personnel, the dashboard transforms raw relational data into meaningful business insights. It showcases proficiency in SQL querying, data transformation, data modeling, DAX, and dashboard design, enabling users to make informed, data-driven decisions through an intuitive and interactive reporting solution.
